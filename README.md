# 🔐 AI-Based Phishing Detection System

This project is a **Machine Learning-based web application** that detects whether a given URL is **safe or phishing**.

It analyzes different features of a URL and provides:
- Risk score 📊
- Detailed analysis 🔍
- AI-based explanation 🧠
- Security recommendations 🔐

---

## 🚀 Features

- 🔍 URL analysis using Machine Learning
- 📊 Risk score visualization (Safe / Suspicious / Phishing)
- 🧠 Explainable AI (feature breakdown)
- ⚡ Real-time detection via web interface
- 🔐 Cyber Security Golden Rules (dynamic tips)

---

## 🛠 Tech Stack

- Python 🐍
- Flask 🌐
- Scikit-learn 🤖
- HTML, CSS, JavaScript 🎨

---

## ⚙️ How It Works

1. User enters a URL in the web interface  
2. The system extracts important features such as:
   - URL length  
   - HTTPS usage  
   - Number of dots  
   - Presence of IP address  
   - Domain structure  

3. These features are passed to a trained **Machine Learning model**  
4. The model predicts:
   - Safe ✅  
   - Phishing 🚨  

5. The result is displayed along with:
   - Risk percentage  
   - Explanation of features  
   - Security suggestions
